## 怎麼玩
1. GDevelop 開 `game.json` → 預覽 ▶ → 標題畫面點「開始冒險」
2. 操作：
   - **方向鍵**移動、**空白鍵**交談/推進劇情
   - **M / Esc** 開選單；選單內 **←→ 切分頁**、↑↓ 選擇、Enter 確定
   - 戰鬥：**方向鍵＋Enter** 或 **滑鼠點擊** 都可以；Esc 返回上一層
3. 流程：**芳蕾鎮開場** → 北出口跟亞倫進礦山（教學戰→魔影劇情戰）→ 回芳蕾鎮（瑪琳入隊）
   → 公會找**緹娜**登錄 → 接委託 → 東之森最東邊討伐**哥布林頭目** → 回報領 200G

## 系統（v3）
- **升級（混合制）**：升級當下只自動加職業基礎屬性；**屬性點(3/級)與技能點(1/級)進入點數池**，
  到 **選單→角色→(選人 Enter)** 分配。升級若習得新技能，勝利畫面會顯示「習得『技能名』！」
- **技能等級**：每個技能可升到 **Lv10**（每級威力 +15%），技能點在角色頁投入
- **五頁選單（M/Esc）**：
  - **角色**：隊列→Enter 進個人頁（立繪＋1/2/3 加屬性點＋↑↓Enter 升技能）
  - **道具**：↑↓選隊員 Enter 用藥水
  - **地圖**：芳蕾鎮區域地圖＋現在位置
  - **稱號**：依進度解鎖（礦山生還者/F級冒險者/哥布林剋星/芳蕾鎮的驕傲），Enter 佩戴後顯示在 HUD
  - **系統**：操作說明、回標題（需按兩次 Enter 確認）
- **隊伍跟隨**：上場隊員在大地圖排成一列跟在領隊後面（序章＝亞倫、第一章起＝瑪琳）
- **對話立繪**：路德/瑪琳/亞倫說話時對話框上方顯示半身像
- **戰鬥**：大尺寸角色圖（不再變形）；鍵盤（方向鍵/Enter/Esc）與滑鼠並行；
  死靈術士會治療同伴、食人魔會橫掃全體；敏捷差產生閃避
- 免費恢復：**旅店（朵拉）**、**神殿（希雅修女）**；吉德首次對話送藥水
