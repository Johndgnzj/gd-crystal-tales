# 水晶戰記 — 內容編輯指南

這個 FF 風回合制戰鬥 demo 的**所有內容都集中在同一個檔案：[`CONTENT.json`](CONTENT.json)**。
你只要改這份檔的數字和文字，再叫我「重建 rpg-battle」，遊戲就會套用你的內容——不用碰任何程式。

---

## 你要怎麼給我內容

**方式 A（推薦）：直接改 `CONTENT.json`**
打開 CONTENT.json，照現有格式改。改完存檔，跟我說一聲，我重跑建置腳本套用。

**方式 B：用試算表或直接打字給我**
如果你比較習慣 Excel／Google 試算表，或想直接在對話裡列給我，也可以——把下面每個表格的欄位填一填傳給我，我幫你轉成 CONTENT.json。

---

## 各區塊欄位說明

### `party` — 我方隊伍（目前 3 人，可增減）
| 欄位 | 意思 |
|------|------|
| `name` | 顯示名字（例：戰士、白魔、黑魔） |
| `sprite` | 用哪個角色圖：`warrior` / `whitemage` / `blackmage`（要換圖再跟我說） |
| `hp` / `mp` | 生命 / 魔力上限 |
| `atk` / `def` / `spd` | 攻擊 / 防禦 / 速度（速度決定出手順序） |
| `skills` | 這角色會的技能 id 清單（對應下面 `skills` 區塊的 id） |

### `enemies` — 敵人圖鑑
| 欄位 | 意思 |
|------|------|
| `name` | 敵人名字 |
| `sprite` | `goblin` / `imp` / `skeleton` / `demon` |
| `hp` / `atk` / `def` / `spd` | 同上 |
| `exp` / `gold` | 打倒後給的經驗值 / 金幣 |

### `skills` — 技能與魔法
| 欄位 | 意思 |
|------|------|
| `id` | 唯一代號（給 party 的 skills 引用） |
| `name` | 顯示名字（例：火焰、治療） |
| `mp` | 消耗魔力 |
| `power` | 威力（傷害或治療量的基準） |
| `target` | `enemy`（單一敵人）/ `ally`（單一隊友）/ `all_enemies`（全體敵人）/ `all_allies`（全體隊友） |
| `kind` | `damage`（傷害）或 `heal`（治療） |
| `element` | 屬性標籤（目前只顯示用，未做屬性相剋——要做再跟我說） |

### `items` — 道具
| 欄位 | 意思 |
|------|------|
| `name` | 道具名（例：藥水） |
| `kind` / `power` | `heal` + 回復量 |
| `count` | 這場戰鬥可用次數 |

### `encounters` — 戰鬥組合
每個組合是一串敵人 id。目前有 `normal` / `hard` / `boss` 三種，對應選單三個按鈕。
要加新戰鬥，多寫一組就行（例：`"cave": ["skeleton","skeleton","imp"]`），再跟我說要不要加到選單。

## 傷害公式（想調整平衡再跟我說）
- 物理攻擊：`(攻擊×2 − 對方防禦) × 隨機0.85~1.0`，1/16 機率會心一擊 ×1.5
- 魔法：`(技能威力 + 施法者攻擊÷2 − 對方防禦÷2) × 隨機0.85~1.0`
- 治療：固定回復技能威力值

## 目前 demo 有的、還沒有的
**有**：選單選戰鬥、速度決定出手序、攻擊/魔法/道具/逃跑、單體與全體魔法、敵人 AI、勝敗結算、魔王戰。
**還沒有（要的話都能加）**：等級與升級、狀態異常（中毒/沉睡）、屬性相剋、多波敵人、戰鬥動畫特效、背景音樂、走地圖遇敵、存檔。
